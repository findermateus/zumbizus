# Zumbizus

Zumbizus é um jogo de sobrevivência desenvolvido na engine GameMaker, com foco em mecânicas de extração, gerenciamento de recursos, interação com NPCs e administração de uma base segura.

No jogo, o jogador enfrenta zumbis em combates corpo a corpo e com armas de fogo, enquanto coleta recursos, explora o ambiente e desenvolve estruturas dentro de sua base. A base funciona como uma zona segura, permitindo a construção de mobiliário e a organização de recursos obtidos durante as expedições.

## Tecnologias Utilizadas

- Engine: GameMaker
- Linguagem: GML (GameMaker Language)
